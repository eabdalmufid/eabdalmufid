Cukup kirim folder website Anda (format ZIP atau Git repo). Kami menerima semua jenis website (HTML statis, PHP, Node.js, Next.js, React, dll) untuk di-online-kan di server kami.

Anda hanya perlu mendesain workflow otomasi di n8n lokal/komputer Anda sendiri, lalu ekspor workflow tersebut menjadi file JSON. Kirim file JSON itu ke kami, lalu kami akan meng-import dan menjalankannya di instance n8n server kami yang online 24 jam.
